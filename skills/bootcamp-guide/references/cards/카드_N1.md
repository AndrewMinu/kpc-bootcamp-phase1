# 주제카드 (참가자용) — [N1] 맛집밀집·볼거리부족 (수도권 상권)

> `[FT]` 영역: B. 콘텐츠 발견·추천 · 난이도: 쉬움 · 이슈 중심(시그니처 N1)
> ★ 지역을 정해주지 않는다. 후보 + 특징을 주고 **참가자가 고른다.**

## 1. 문제 중심 주제(이슈)
*맛집은 넘쳐나는데 정작 볼거리가 약한 도시다. 밥 먹으러는 와도 딱히 '여기서 시간 보낼 이유'가 없어서 관광 목적지로는 안 굳어진다. 골목·로컬 콘텐츠를 발굴하거나 미식에 체험을 붙여서 '머물 이유'를 만들어주는 게 포인트다.*

## 2. 대상 사용자/페르소나
맛집 투어로 왔지만 밥만 먹고 딱히 더 볼 게 없어 금방 떠나는 여행자. / 유동인구는 있는데 체류로 안 이어지는 상권 상인.

## 3. 검증 전 가설
음식 공급은 풍부한데 볼거리(관광지)가 약해, 미식 방문이 체류·재방문으로 이어지지 못할 수 있다.

## 5. 확인하고 싶은 단서 (시그니처)
음식점 밀도는 상위인데 관광지·콘텐츠가 약하고 체류시간이 짧으면 → 머물 이유가 부족하다는 근거.

## 6. 예상 아웃풋 이미지
맛집을 중심으로 주변 골목·체험·포토스팟을 붙여 '먹고 노는' 반나절 코스를 만드는 화면.

## 7. 추천 MVP
관심 음식 선택 → 주변 로컬 콘텐츠 매칭 → 미식+체험 코스 → 발표용 골목 지도.

# 주제카드 (참가자용) — [A2] 자원 단조로움 (다양성 낮음·날씨/계절 취약)

> `[FT]` 영역: B. 콘텐츠 발견·추천 · 난이도: 어려움 · 이슈 중심(시그니처 A2)
> ★ 지역을 정해주지 않는다. 후보 + 특징을 주고 **참가자가 고른다.**

## 1. 문제 중심 주제(이슈)
*관광 자원이 한두 종류에 쏠려 있어서 구성이 단조로운 곳이다. 날씨가 궂거나 비수기가 되면 대신 갈 만한 다른 종류가 없어서 관광이 특정 조건에서만 굴러간다. 부족한 유형을 보강해서 사계절 어느 때나, 어떤 날씨에도 갈 곳이 있게 다양화하는 게 과제다.*

## 2. 대상 사용자/페르소나
날씨 궂거나 비수기에 갔더니 할 게 하나뿐이라 실망한 여행자.

## 3. 검증 전 가설
관광 공급이 한두 유형에 쏠려 단조로워, 비수기·악천후에 대안이 없고 방문이 특정 조건에 종속될 수 있다.

## 5. 확인하고 싶은 단서 (시그니처)
공급 유형 집중도(HHI)가 높고 방문이 특정 계절·날씨에 쏠리면 → 자원 단조로움 근거.

## 6. 예상 아웃풋 이미지
부족한 유형을 보강해 사계절·전천후로 즐길 대안을 제안하는 화면.

## 7. 추천 MVP
현재 자원 구성 확인 → 부족 유형 보강 아이디어 → 사계절 코스 → 발표용 다양화 카드.

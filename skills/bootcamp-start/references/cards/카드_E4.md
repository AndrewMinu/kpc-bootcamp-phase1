# 주제카드 — [E4] 아웃도어·캠핑 — 뜨는 수요를 자원이 못 받는다

> `[FT]` 그룹: **E · 대상** · 영역: A. 이동·동선 · 난이도: 쉬움
> ★ 지역을 정해주지 않는다. 후보와 특징만 주고 **참가자가 고른다.**

## 1. 문제 중심 주제(이슈)
*레포츠·자연 자원이 풍부한 곳인데 캠핑 인프라가 그만큼 따라오지 못한 곳이다. 러닝·등산·서핑·캠핑 같은 아웃도어는 한 번 빠지면 계속 다시 오고 오래 머물러서 지역에 특히 유리한 수요다. 난이도·계절별 액티비티에 장비·강습·숙박을 묶고 동호회처럼 재방문을 만드는 방향이 좋다.*

## 2. 대상 사용자/페르소나
러닝·등산·서핑·캠핑을 찾아 반복해서 떠나는 아웃도어 여행자.

## 3. 검증 전 가설
레포츠 자원은 풍부한데 캠핑 인프라가 못 따라온다. 재방문·장기체류가 강한 아웃도어 수요를 못 받고 있을 수 있다.

> **검증 시(2단계)**: 고른 지역을 라이브로 다시 받아 사슬(방문→체류→소비)을 확정한다.

## 5. 확인하고 싶은 단서 (시그니처)
레포츠 공급은 상위인데 캠핑장이 하위면 → 아웃도어 수용력 공백 근거.

## 6. 예상 아웃풋 이미지
난이도·계절별 액티비티에 장비·강습·숙박을 묶어 제안하는 화면.

## 7. 추천 MVP
활동·수준 선택 → 코스·장비·야영 매칭 → 액티비티 패키지 → 발표용 활동 지도.
